# 1. Overall System Architecture

Your system will have **two main parts**:

1️⃣ **Frontend (React UI)**
2️⃣ **Backend (AI processing pipeline)**

Pipeline flow:

```text
User enters article / URL
        ↓
React Frontend
        ↓
FastAPI Backend
        ↓
Gemini Claim Extraction
        ↓
spaCy Entity Extraction
        ↓
Fact-check APIs
        ↓
Credibility Score Engine
        ↓
Results returned to React UI
```

---

Use:

**Axios**

```javascript

```

# 3. Backend Stack

Framework:

**FastAPI**

Why FastAPI:

* very fast APIs
* async processing
* great for AI pipelines
* automatic documentation

Example endpoint:

```
POST /analyze
```

Input:

```json
{
 "article": "text of the article"
}
```

Output:

```json
{
 "claims": [...],
 "credibility_score": 47
}
```

# 4. AI Model
